# Changes Log Governance

## Access Policy

- **Read**: Mandatory "Read-Before-Write" for context.
- **Write**: Append-only.
- **Delete**: Forbidden (except for sensitive data scrubbing).

## Structure

- Format: `YYYY/MM/DD/HHMMSSZ_<actor>_<action>.md`
- Must update `INDEX.md`
- Committed to git.
